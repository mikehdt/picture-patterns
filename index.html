<picture> patterns

A collection of code patterns and demonstrations using srcset combined with the HTML <img> and <picture> elements

The basics

Coming soon… just know for now that the <picture> element is a fussy and inconsistently implemented beast, and that the awesome Picturefill makes things bearable. Note that Picturefill is not used with any of the examples yet (it will be available as an option in a later revision), so you are at the mercy of your current browser…

This project is a work in progress. More examples and resources will be added soon.
